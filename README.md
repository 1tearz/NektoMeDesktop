# NektoMeDesktop

Удобная десктопная обёртка для голосового чата [nekto.me](https://nekto.me) на базе Electron. Добавляет множество функций, отсутствующих в браузерной версии: регулировка громкости собеседника, выбор микрофона, смена темы оформления, горячие клавиши, кастомное изображение и блокировка рекламы.

### Готовые сборки
В разделе [Releases](https://github.com/1tearz/NektoMeDesktop/releases) доступны установщики для Windows (exe) и пакеты для Linux (deb). Скачайте и запустите.

## Возможности

- **Громкость собеседника** — отдельный ползунок, позволяющий регулировать громкость голоса оппонента (работает через Web Audio API).
- **Выбор микрофона** — можно выбрать конкретное устройство из списка, даже если сайт запрашивает «микрофон по умолчанию».
- **Смена темы** — несколько встроенных тем (winter, red, white, forest-wood) и возможность загрузить свою CSS-тему.
- **Горячие клавиши** — управление всеми основными действиями без мыши:
  - `F1` — начать разговор
  - `F2` — завершить разговор (вызов кнопки «Завершить»)
  - `F3` — блокировка/разблокировка микрофона (внутри сайта, только после минуты общения)
  - `F4` — искать нового собеседника
  - `F5` — отменить в диалоге подтверждения
  - `F6` — подтвердить завершение в диалоге
- **Кастомное изображение** — замена стандартной картинки в нижней части окна на своё (автоматически подгоняется под размер 600×290).
- **Блокировка рекламы** — встроенный фильтр запросов к рекламным доменам.
- **Удаление навязчивых элементов** — скрываются ссылки на мобильные приложения и блок с описанием.
- **Кнопка настроек** — всегда доступна в правом верхнем углу для быстрого доступа к настройкам.
- **Мониторинг памяти** — в окне настроек отображается потребление RSS.
- **Персистентность настроек** — все настройки (тема, громкость, микрофон, изображение) сохраняются между запусками.

## Установка

### Сборка из исходников
1. Убедитесь, что установлены [Node.js](https://nodejs.org/) и npm.
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/tearz/nekto-desktop.git
   cd nekto-desktop

Установите зависимости:
bash

npm install

Запустите приложение:
bash

npm start

    Для сборки установщиков:

        Windows: npm run build:win

        Linux: npm run build:deb

Использование

После запуска откроется главное окно с сайтом nekto.me/audiochat#. Кнопка настроек (⚙) в правом верхнем углу открывает окно конфигурации. Там можно:

    Выбрать тему оформления.

    Настроить громкость собеседника (от 0 до 100%).

    Выбрать микрофон (если доступ не разрешён, будет предложено запросить доступ или открыть настройки системы).

    Импортировать кастомную тему (CSS-файл).

    Заменить изображение на своё.

    Посмотреть текущее потребление памяти.

Горячие клавиши работают глобально внутри окна приложения.
Технические детали

    Приложение построено на Electron с использованием contextBridge и preload-скриптов для безопасного взаимодействия с веб-страницей.

    Управление громкостью реализовано через единый AudioContext и GainNode, что позволяет менять громкость независимо от плеера сайта.

    Перехват микрофона происходит путём подмены getUserMedia и замены треков в уже существующих RTCPeerConnection.

    Темы применяются через insertCSS и могут быть сохранены в папке пользователя.
